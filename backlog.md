## 已知非阻断观察（不立刻开工）

### BO-1 · 本地启发式对中文顿号列举识别不一致

- **现象**：对话模式 `基础语法、函数、类、装饰器` 只识别出 1 个主分支；IDE 协同模式 `东京、京都、大阪` 能识别出 4 个分支
- **来源**：第五波产品验收（Trae 发现）
- **影响**：不配模型时，本地引擎的导图质量在两种模式下不一致
- **缓解**：配置 LLM（OpenAI / Anthropic / ccswitch）即可绕过本地引擎
- **优先级**：低（配模型即改善）
- **位置**：`api/services/heuristicEngine.ts`（splitSentences / extractListItems 的边界条件）

### BO-2 · 畸形 JSON 请求体返回 500

- **现象**：向任意 API 发送非法 JSON body → express.json 解析失败 → 500
- **来源**：第五波产品验收（Trae 发现）
- **影响**：合法调用方不受影响；仅当客户端发错 body 时暴露 500
- **优先级**：低（边缘场景）
- **位置**：`api/app.ts`（可在 express.json 后加解析错误中间件，返回 400）

### BO-3 · 重新生成 SSE 的 `ERR_ABORTED` 日志噪音

- **现象**：重新生成的 SSE 请求在浏览器 network log 显示 `ERR_ABORTED`
- **来源**：第五波产品验收（Trae 发现）
- **影响**：导图实际正常重生成，无 UI 报错；仅 network log 有噪音
- **优先级**：低（不影响功能）
- **位置**：`api/routes/mindmap.ts` regenerate + `src/store/useStore.ts` regenerate（流正常关闭的副作用）
